### В чому різниця між реалізованими структурами, в чому особливості кожної структури?

* **Стек (Stack):** Це лінійна структура даних, яка працює за фундаментальним принципом LIFO (Last In, First Out). Головна особливість полягає в тому, що всі операції додавання (`push`) та вилучення (`pop`) відбуваються лише з одного кінця структури (вершини).
* **Циклічний журнал повідомлень (Circular Buffer):** Це структура на основі зв'язного списку фіксованого розміру, призначена для запису текстових повідомлень та часу їх надходження. Її ключова особливість — поведінка при переповненні: коли журнал заповнюється повністю, кожен новий запис автоматично замінює (перезаписує) найдавніше збережене повідомлення.
* **Розгорнутий зв'язний список (Unrolled Linked List):** Це гібридна структура, що поєднує властивості масивів та класичних зв'язних списків. Замість того, щоб зберігати по одному елементу в кожному вузлі, вузли містять масиви елементів. Ця структура підтримує стандартні операції списків (обхід в обох напрямках, доступ за індексом, видалення та додавання в довільні позиції), але завдяки групуванню елементів значно зменшує накладні витрати пам'яті на вказівники та покращує використання кешу процесора.

### Алгоритмічна складність операцій для кожної реалізованої структури

* **Stack:**
  * `stack_create`, `stack_is_empty`: $O(1)$ 
  * `push` (додавання): $O(1)$ 
  * `pop` (вилучення): $O(1)$ 
  * `peek` (доступ до вершини): $O(1)$ 

* **Циклічний журнал повідомлень (Circular Buffer):**
  * Додавання повідомлення (із можливою заміною старого): $O(1)$
  * Отримання останніх K повідомлень: $O(K)$

* **Unrolled Linked List:**
  * Доступ за індексом (`get` / `set`): $O(N)$ у найгіршому випадку (де $N$ — кількість блоків), проте на практиці працює значно швидше за звичайний зв'язний список завдяки переходам одразу цілими масивами
  * Пошук за значенням: $O(N)$
  * Додавання елемента на початок / кінець: $O(1)$
  * Додавання / видалення елемента за довільною позицією: $O(N)$ для лінійного пошуку потрібного блоку, плюс зсув елементів всередині самого масиву блоку за $O(1)$ (оскільки розмір масиву фіксований константою)
